# Встановлення (Windows)

## Швидкий старт — без Python

1. Завантажте **`EAW_FOC_Tools_v1.0.0_Windows.zip`** з [Releases](https://github.com/alex0927-cmd/Trainer-and-Save-Editor-for-Game-Star-Wars-Empire-at-War---Forces-of-Corruption/releases).
2. Розпакуйте в будь-яку папку (наприклад `C:\Games\EAW_FOC_Tools\`).
3. Запустіть гру **Forces of Corruption** (`swfoc.exe`).
4. **Трейнер:** ПКМ на `Запуск_Трейнера.bat` → **Запуск від імені адміністратора**.
5. **Редактор:** `Запуск_Редактора.bat` (адмін не обов'язковий).

> Якщо Windows Security блокує `.exe` — використовуйте **bat-файли** (вони запускають Python-версію, якщо встановлено Python).

## Що в архіві

| Файл | Призначення |
|------|-------------|
| `EAW_FOC_Trainer.exe` | Трейнер +10 (чити лише для гравця) |
| `EAW_FOC_Editor.exe` | Редактор XML гри |
| `Запуск_Трейнера.bat` | Рекомендований запуск трейнера |
| `Запуск_Редактора.bat` | Запуск редактора |
| `README.md` | Короткий опис |
| `INSTALL.md` | Цей файл |

## Трейнер — перші кроки

1. Підключитися до `swfoc.exe`.
2. **Авто-зв'язати** кредити (2 рази: змініть суму в грі між натисканнями).
3. Введіть «Бажане» → **Застосувати** → увімкніть чит.

Повна документація: папка `docs\` у репозиторії на GitHub.

## Вимоги

- Windows 10/11
- Star Wars: Empire at War — **Forces of Corruption** (колекція / Steam / GOG)
- Трейнер: права **адміністратора**
- Шлях до гри: за замовчуванням шукається `...\Star Wars Empire At War Collection\data2\`

## Антивірус

Трейнер читає пам'ять гри — Defender може видати попередження. Додайте папку у винятки або використовуйте `Запуск_Трейнера.bat`.
