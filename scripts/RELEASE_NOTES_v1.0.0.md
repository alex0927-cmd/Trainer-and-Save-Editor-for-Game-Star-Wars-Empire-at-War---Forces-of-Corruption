## Завантаження

Розпакуйте **EAW_FOC_Tools_v1.0.0_Windows.zip** у будь-яку папку.

## Швидкий старт

1. Запустіть гру **Forces of Corruption** (`swfoc.exe`)
2. **Запуск_Трейнера.bat** → ПКМ → **Запуск від імені адміністратора**
3. Підключитися → Авто-зв'язати кредити (2 кроки) → Застосувати → увімкнути чит
4. Редактор: **Запуск_Редактора.bat**

## У складі

- `EAW_FOC_Trainer.exe` — трейнер +10 (лише для гравця)
- `EAW_FOC_Editor.exe` — редактор XML
- `Запуск_Трейнера.bat` / `Запуск_Редактора.bat`
- `README.md`, `INSTALL.md`
