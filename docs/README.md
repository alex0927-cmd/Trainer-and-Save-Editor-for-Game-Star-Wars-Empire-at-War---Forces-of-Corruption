# Документація проєкту EAW FOC Modding & Tools

Повний технічний звіт про те, **як було досліджено гру**, **які рішення прийнято** і **чому саме так** реалізовано всі модифікації та інструменти.

**Останнє оновлення:** червень 2026 — додано авто-зв'язування, гарячі клавіші, файлове логування.

---

## Для кого цей документ

- Гравець, який хоче зрозуміти, що саме змінено в його установці.
- Моддер, який продовжить роботу над XML, MEG-архівами або трейнером.
- Розробник, який захоче переписати/покращити `EAW_FOC_Editor.exe` та `EAW_FOC_Trainer.exe`.

## Два підходи до «читів»

### Підхід A — XML-моддинг

- Файли в `data2\Data\XML\` **перекривають** `Config.meg`.
- Зміни **глобальні** (впливають на ШІ).
- Плюси: стабільно, без адмінправ.
- Мінуси: не «тільки для гравця».

### Підхід B — трейнер (RAM)

- Під час гри читає/записує пам'ять `swfoc.exe`.
- Авто-зв'язування знаходить **ваш** гаманець.
- Гарячі клавіші, live-значення, файловий лог.
- Плюси: player-only, без змін файлів гри.
- Мінуси: адмін, калібрування, AoB залежить від версії exe.

## Швидкий старт — трейнер

```
1. Запустити swfoc.exe
2. Запуск_Трейнера.bat (від адміна)
3. Підключитися
4. Авто-зв'язати кредити (2 кроки: змінити в грі між натисканнями)
5. Ввести «Бажане» → Застосувати → увімкнути чит
6. (Опційно) ⌨ — призначити гарячі клавіші
7. (Опційно) ☑ Записувати все у файл — для діагностики
```

Повний посібник: [09-TRAINER-DETALNO.md](09-TRAINER-DETALNO.md).
