# 13. Обмеження, ризики та рекомендації

## Обмеження XML-модів

| # | Обмеження |
|---|-----------|
| 1 | HP×99999 застосовується до **усіх** фракцій |
| 2 | ШІ отримує той самий дохід планет, якщо захопить їх |
| 3 | Зміни в `FACTIONS.XML` впливають на **опис фракції**, не на «тільки слот 0» |
| 4 | Герої в **кампанії** можуть загинути через **Lua-сюжет** |
| 5 | `Build_Cost_Credits` у сотнях `UNITS_*.XML` **не змінені** — юніти коштують грошей (але їх багато) |
| 6 | Клієнт стає **non-vanilla** — мультиплеер з іншими гравцями проблемний |
| 7 | Потрібен **перезапуск** гри після XML-змін |

---

## Обмеження трейнера

| # | Обмеження |
|---|-----------|
| 1 | Потрібен запуск **від адміністратора** |
| 2 | **32-bit** `swfoc.exe` — AoB-патерни з 64-bit CE-таблиць можуть **не збігтися** |
| 3 | Після **патчу гри** Steam патерни можуть зламатися |
| 4 | Калібрування кредитів/HP — **ручний крок**, інакше чит не працює |
| 5 | God Mode через HP-freeze — лише для **відкаліброваних** адрес (обмежена кількість юнітів) |
| 6 | `instant_build` / `free_build` без знайденого AoB — **не активуються** |
| 7 | Антивірус / SmartScreen може блокувати exe — використовуйте `Запуск_Трейнера.bat` |
| 8 | Гарячі клавіші потребують **адміна** (бібліотека `keyboard`) |
| 9 | Адреси RAM у `trainer_calibration.json` **невалідні** після перезапуску гри |
| 10 | Live-значення `—` без попереднього авто-зв'язування |
| 11 | Не для **онлайн-мультиплееру** |

---

## Відомі конфлікти

### XML HP + Trainer God Mode одночасно

Подвійний ефект — надлишково, але не ламає гру. Рекомендація: для чесного ШІ вимкніть XML HP (`1.0` / `1.5`) і використовуйте лише трейнер.

### Попередні XML-чити в data2

Якщо залишились `99999` у `GAMECONSTANTS`, ШІ **дуже живучий** незалежно від трейнера. Для балансу відновіть ваніль через редактор.

## Юридична та етична примітка

- Модифікація **одиночної** гри для власного задоволення — типова практика спільноти.
- Розповсюдження exe третім особам — на ваш ризик (ліцензія LucasArts / Disney).
- Не використовуйте в ranked / multiplayer.

## Як звітувати про проблему

При збої вкажіть:

1. Версію гри (Steam / GOG / CD).
2. Розмір і дату `swfoc.exe`.
3. Режим (GC / Skirmish / Campaign).
4. Текст з **журналу трейнера** або файл `trainer.log` (якщо увімкнено логування).
5. Чи є змінені XML у `Data\XML\`.

## Повернення до ванілі

### Повністю

```
1. Перейменувати data2\Data\XML → XML_disabled
2. Або видалити змінені XML — гра візьме Config.meg
```

### Частково

Редактор → **Відновити ваніль** → **Зберегти**.

### Трейнер

Просто закрити — патчі відкатяться (якщо процес не вбито жорстко).
